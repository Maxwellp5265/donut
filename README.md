## 🍩 What this app does

donut shows a moving donut in your terminal window. It uses Go, Bubble Tea, and Lip Gloss to draw clean ASCII art with steady motion and low CPU use.

This app is for Windows users who want a small tool that runs in a terminal window. After you download it, you open it and see the animation right away.

## 💻 How to install on Windows

Follow these steps:

1. Open the download link in your browser.
2. Find the Windows version of donut.
3. Download the file to your Downloads folder.
4. If the file is in a ZIP archive, right-click it and choose Extract All.
5. Move the app file to a folder you can find again, such as Desktop or Documents.
6. If Windows shows a security prompt, choose Run or More info, then Run anyway if you trust the source.
7. Open PowerShell or Windows Terminal.
8. Go to the folder where you saved donut.
9. Run the app file.

If the file name ends in `.exe`, double-click it to start. If it opens in a terminal window, keep that window open while the animation runs.

## 🧩 Troubleshooting

### The app does not open

Try these steps:

- Re-download the file
- Check that the file is not blocked by Windows
- Move it to a simple folder path, such as `C:\Tools\donut`
- Run it again from PowerShell or Windows Terminal

### The animation looks broken

Try these steps:

- Use Windows Terminal
- Increase the terminal width
- Change the font to one that supports line art
- Turn off very old console settings
- Open a new terminal session and run the app again

### The colors do not show

Try these steps:

- Use a modern terminal app
- Check terminal color settings
- Update Windows Terminal if you use it
- Make sure your terminal theme does not hide color output
